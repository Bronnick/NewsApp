**Technology stack:**
+ Kotlin
+ View binding
+ Jetpack
  + Datastore
  + Navigation component
+ Retrofit for networking
+ Kotlin Coroutines for asynchronous work
+ Coil for loading images asynchronously
+ Paging library
+ Hilt for dependency injection
+ Architecture
  + MVVM
  + Repository pattern
